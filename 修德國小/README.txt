# Sigma.js校際合作網絡展示

這個資料夾提供可直接開啟的Sigma.js互動網絡頁。

## 檔案
- `index.html`：主展示頁，已內嵌GEXF資料
- `network.gexf`：原始GEXF檔

## 使用方式
### 本機直接開啟
直接用瀏覽器開啟`index.html`即可。

### 部署到GitHub Pages
把整個資料夾上傳到GitHub repository後，啟用Pages即可。

## 功能
- 顯示有向網絡箭頭
- 機構方形、成員圓形
- 拖曳節點
- 搜尋與篩選
- 顏色與大小控制
- 匯出PNG

## 注意
此頁面使用CDN載入Sigma.js、Graphology與方形節點renderer，因此瀏覽器需可連網。
